### 情報理論と符号化

#### 情報量
**情報量** *I (J) = -log(2)P(J)*
J：事象
P(J)：事象Jの起こる確率

**平均情報量** *H=Σ{P(J)×I(J)*

####　情報源符号化
**ハフマン符号化**：木で確率に応じてビット化
**ランレングス符号化**：連続を数字で表現

####　ディジタル符号化
**パルス符号変調（PCM: Pulse Code Modulation）**
1. 標本化
2. 量子化
    - 量子化ビット数：8ビットで0-255の数値に変換
3. 符号化 

**標本化定理**：サンプリング周波数を上限周波数の2倍とする

*ディジタルデータ量/s = サンプリング周波数 × 量子化ビット数*
